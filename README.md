# VU "Dynamic Programming and Reinforcement Learning" Week 2 Assingment

## Setup
This repository uses [pyenv](https://github.com/pyenv/pyenv) and [poetry](https://github.com/python-poetry/poetry) for python version and dependency management. Follow thei individual instructions to set them up.

(optional) install the locked python version via pyenv:
```bash
cd vu_dp_rl_week1_assingment
pyenv install
```

install the dependencies via poetry:
```bash
poetry install
```

install the pre-commit hooks (for automatic code formatting):
```bash
pre-commit install
```

## How to run
Once the `setup` is complete, you can simply run the `main.ipynb` notebook to generate the reported insights and artifacts!
